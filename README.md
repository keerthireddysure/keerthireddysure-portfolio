# Keerthi Reddy Sure — Portfolio v2

Static interview-ready portfolio website.

## How to run
Open `index.html` in a browser.

## How to deploy
Upload the folder to Vercel, Netlify, GitHub Pages, or any static hosting service.

## Structure
- `index.html` — main page
- `styles.css` — styling
- `script.js` — interactive skills, experience, and case study modals
- `assets/` — profile and case study diagrams
